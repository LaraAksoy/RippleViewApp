# RippleView

RippleView is a dynamic SwiftUI application that allows users to create interactive visual stories by placing landmark stickers on a scrollable canvas. Designed with a focus on fluid gestures and playful animations, this app demonstrates the power of SwiftUI's animation and gesture systems.

## âœ¨ Features

- **Interactive Canvas**: users can tap or drag anywhere on the screen to place stickers.
- **Fluid Animations**: Each placed item appears with a custom spring animation, creating a delightful "pop" effect.
- **Randomized Assets**: Automatically cycles through a curated set of landmark images for variety.
- **Infinite Scrolling**: A vertically scrollable area allows for extensive creativity without space constraints.
- **Gesture Driven**: seamless integration of drag gestures for real-time interaction.
